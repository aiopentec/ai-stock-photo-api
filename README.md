# AI Stock Photo API

Free AI-generated stock photography platform.

## Quick Start

1. Add HF_TOKEN to GitHub Secrets
2. Push to GitHub
3. Enable GitHub Pages
4. Site goes live!

## API Endpoints

- GET /api/images - Browse all images
- GET /api/info - API statistics
- GET /api/search?q=keyword - Search images
